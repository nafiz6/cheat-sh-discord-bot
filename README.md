# Cheat-sh
Simply curls cheat.sh and returns the message in a discord bot

# REQUIREMENTS
discord.py - `pip install -U discord.py`
requests - `python -m pip install requests`
dot-env - `pip install -U python-dotenv`

To run
`python bot.py`
